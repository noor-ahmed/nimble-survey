## Usage

You will need `node` and `npm` installed globally on your machine.  Built using `node-14.6.0` and `npm-6.14.6`

Install packages
```
npm install
```

Start server
```
npm start  
```
Run test suite
```
npm test
```
Run ESlint
```
npm run lint
```

Locally Port: [localhost:3000](localhost:3000)
